Projects

Down in the DM

Down in the DM is an instant messenger application developed with Python and Flask. Once users register they are added to the local network and have the ability to message any other created users as well as view their inbound messages in order by which they were sent.

Source Code

Let's Hoop!

Let's Hoop is a Java web application developed to solve a problem that most people who like to play basketball face; not knowing if there are runs or not. This web application allows for users to create, join, and make modifications to upcoming basketball open runs. Additionally, for those who have joined a mutual event there is a message board for any instant notifications such as park services or the court being heavily occupied.

Source Code

The Great Cake Rate

The Great Cake Rate is an Angular application that allows users to Submit their own cake designs and recieve reviews and ratings from other users about their product. This application somewhat serves as the 'Yelp' of rating delicious cakes.
